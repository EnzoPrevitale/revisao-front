# Revisão Front

Este repositório utiliza MongoDB como banco de dados. Instale aqui: https://fastdl.mongodb.org/windows/mongodb-windows-x86_64-8.2.4-signed.msi
Instale também o MongoDB Shell: https://downloads.mongodb.com/compass/mongosh-2.6.0-x64.msi

Execute <code>mongosh</code> no PowerShell.

No mongosh, rode os comandos em mongosh.txt.

Execute o back-end e rode o front.
